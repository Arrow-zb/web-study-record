# 定制PM2镜像
PM2守护进程
```json
apps:
 - script : app.js
instances: 2
watch : true
env :
NODE_ENV: production
```

```dockerfile
FROM keymetrics/pm2:latest-alpine 
WORKDIR /usr/src/app 
ADD . /usr/src/app 
RUN npm config set registry https://registry.npm.taobao.org/ &&  npm i 
EXPOSE 3000 
#pm2在docker中使用命令为pm2-docker 
CMD ["pm2-runtime", "start", "process.yml"]
```
