# 字符串的扩展

## 1. 字符的 Unicode 表示法

ES6 中只要将码点放入大括号，就能正确解读字符

js中有6种方法可以表示一个字符

```js
'\z' === 'z'  // true
'\172' === 'z' // true
'\x7A' === 'z' // true
'\u007A' === 'z' // true
'\u{7A}' === 'z' // true
```
