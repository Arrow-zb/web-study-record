# 8. 用户管理工具

## 8.1 用户

### 1. 添加用户

```bash
useradd -m username
```

该命令为用户创建响应的账户和用户目录 `/home/username`

用户添加之后，设置密码：

密码以交互方式创建：

```bash
passwd username
```

### 2. 删除用户

```bash
userdel -r username
```

不带选项使用 userdel 只会删除用户。用户的目录仍将会在 `/home`目录下。要完全的删除用户信息，使用 `-r` 选项；

账号切换，登录账号为 userA 用户状态下，切换到 userB用户账号工作

```bash
su userB
```

进入交互模型，输入密码授权进入
