# 鉴权之单点登录

## 1. 前言

单点登录（Single Sign On）是一种身份认证方案，允许用户使用单个 ID 和密码登录到多个相关联但独立的软件系统的任何一个，并且用户只需要登录一次即可访问其它（相关联）的系统。

在不同的业务场景下，采用的方案也不尽相同；一般主要分两大类：

1. 同域下单点登录，这种情况下基于顶域的session-cookie即可实现。
2. 不同域下单点登录，一般可采用两种方式。
   - 基于 JSONP 实现
   - 基于 CAS  实现
